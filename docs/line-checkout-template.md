# 退房感謝 LINE 訊息模板

> 使い方：チェックアウト後、LINE でゲストに送信。Google 評論とサウナイキタイのリンクを含む。

---

## テンプレート（コピペ用）

```
🐱 ご宿泊ありがとうございました！

和心村にお越しいただき、ありがとうございます。
猫たちもまた会えるのを楽しみにしています 🐾

もしよろしければ、ご感想をお聞かせください。
1分で完了します 🙏

📝 Google レビュー:
https://g.page/r/[PLACE_ID]/review

♨️ サウナイキタイで「いきたい」:
https://sauna-ikitai.com/saunas/[施設ID]

またのお越しをお待ちしております！
和心村スタッフ一同
```

---

## カスタマイズのポイント

1. **[PLACE_ID]** → Google Business Profile の Place ID に置き換え
   - GBP管理画面 → 「レビューのリンクを取得」でURLを確認
2. **[施設ID]** → サウナイキタイに登録後、施設ページの URL から取得
3. 送信タイミングは **チェックアウト当日の夕方〜翌日** がベスト（体験が新鮮なうちに）
4. **写真付きレビュー**をお願いすると、Google での表示順位が上がる

---

## QR コード用 URL

Google レビュー直リンクを QR コード化して退房処に設置：
- QR生成サイト: https://www.qrcode-monkey.com/
- URL: `https://g.page/r/[PLACE_ID]/review`
- サイズ: A5（210×148mm）くらいが見やすい
- デザイン案: 猫のイラスト + 「ご感想をお聞かせください 🐱」

---

## 注意事項

- Google のポリシー上、「★5をつけてください」等の**評価の誘導は禁止**
- 「ご感想をお聞かせください」はOK
- **ネガティブなレビューにも必ず返信**する（放置すると印象が悪い）
- 返信テンプレート例:
  - 高評価: 「ありがとうございます！猫たちも喜んでいます 🐱 またのお越しをお待ちしております」
  - 低評価: 「貴重なご意見ありがとうございます。ご不便をおかけし申し訳ございません。改善に努めてまいります。具体的なご要望がございましたら info@washinmura.jp までお知らせください」
