## SEO 効果の期待値

| 施策 | 効果 |
|------|------|
| blog 間の相互リンク | 滞在時間増加、回遊率向上 |
| blog → guide ページ | guide 全体の権威性向上 |
| guide → Strikingly | 既存 120+ 記事の SEO パワーを guide に流す |
| Strikingly → guide | 新しい blog 記事にドメイン権威を付与 |

**重要:** 相互リンクのアンカーテキストにはキーワードを含める。
- ❌ 「こちら」「詳しくはこちら」
- ✅ 「築200年の古民家サウナ体験の詳細」「16匹の保護猫と泊まれるグランピング」
